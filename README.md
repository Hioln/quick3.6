# quick-cocos2d-x v3.6

*新功能
1.使用Luajit2.1支持arm64
2.集成iosIAP
3.CCSUIloader2完善中


*使用方法
1.安装 
setup.py

2.创建工程 
cocos new MyGame -p com.cocoslua.mygame -l lua [-d NEW_PROJECTS_DIR]

3.编译运行
win
cocos run -p win32 

android
cocos run -p android -j 4

mac
cocos run -p mac

ios
cocos run -p ios

*其他
UI编辑器使用Cocos v2.1
